# Diabetes Risk Prediction Using Machine Learning

## Overview
This project predicts whether a person is at risk of diabetes using machine learning techniques and patient health data.

## Dataset
- Pima Indians Diabetes Dataset

## Features Used
- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

## Project Workflow
- Data Cleaning
- Missing Value Handling
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Feature Scaling
- Model Training
- Model Evaluation

## Models Used
- Logistic Regression
- KNN
- Decision Tree
- Naive Bayes
- SVM
- Random Forest
- Extra Trees
- Gradient Boosting
- AdaBoost
- LightGBM

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC

## Tools & Libraries
- Python
- Pandas
- NumPy
- Scikit-Learn
- Seaborn
- Matplotlib
- Plotly
- LightGBM

## Results
Multiple machine learning models were trained and compared using cross-validation to identify the best-performing model for diabetes prediction.
